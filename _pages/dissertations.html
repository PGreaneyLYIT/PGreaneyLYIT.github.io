--- layout: archive title: "Dissertation Topics" permalink: /dissertations/ author_profile: true --- I am available to supervise a limited number of dissertations on the MSc programmes in Data Science and BDA & Artificial Intelligence. If you are interested in working under my supervision, you should contact me in good time, ahead of the project proposal deadline. A list of dissertations I've previously supervised is available here.

One thing worth considering is aligning your dissertation topic with the United Nations Sustainable Development Goals.

Some topics which would be of interest:
  • Finding solutions to differential equations using deep learning
  • Applications of neural differential equation models
  • Crop yield prediction using climate data
  • Forecasting and tracking air quality
  • Applications of machine learning to transport, including traffic forecasting, public transport demand and optimisation
  • Sentiment analysis for public and political opinion
  • Effects of annotation quality on ML model performance

Papers with Code is a useful repository of machine learning papers together with their with code implementations. Some useful sources of data include:
  • Ireland's Open Data Portal
  • The EU's European Data Portal
